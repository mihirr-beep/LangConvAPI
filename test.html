A throwaway page for debugging your /translate endpoint before you wire it into ChatGPT.
⚠ Opened from file:// — browsers block network calls from file:// origins, so this page can pick languages and a file but cannot reach the API. To actually test translation: start the server (uvicorn main:app --port 8000) and reopen this page at http://localhost:8000/test (FastAPI serves the same HTML at that URL — same code, but properly same-origin).
